Утилита фильтрации содержимого файлов.

ВЕРСИЯ JAVA: java 25.0.1 (минимальная версия для запуска утилиты: java 17)

ВЕРСИЯ GRADLE: gradle 9.0.0

Для запуска необходимо:

  1. Скачать и распаковать утилиту;
  2. Открыть папку с распакованным проектом через терминал;
  3. Ввести ./gradlew build и дождаться окончания установки;
  4. Ввести java -jar build/libs/data-classifier-1.0.0-all.jar [с различной комбинацией указания флагов и путя откуда происходит чтение].

Пример команды запуска:

java -jar build/libs/data-classifier-1.0.0-all.jar in1.txt in2.txt -f -a -p sample- -o \tmp

Описание флагов и специфика их реализации:

 -o: Задает путь, куда будут сохранятся файлы с успешно обработанными данными. ЧТОБЫ ФЛАГ КОРРЕКТНО СРАБОТАЛ, НЕОБХОДИМО УКАЗАТЬ ПУТЬ НАЗНАЧЕНИЯ СРАЗУ ПОСЛЕ ФЛАГА;
 -p: Задает префикс исходящих файлов. ЧТОБЫ ФЛАГ КОРРЕКТНО СРАБОТАЛ, НЕОБХОДИМО УКАЗАТЬ ПРЕФИКС СРАЗУ ПОСЛЕ ФЛАГА;
 -a: Режим добавления в уже существующие файлы (по умолчанию данные в файле перезаписываются);
 -s: Краткая статистика (содержит количество элементов записанных в исходящие файлы);
 -f: Полная статистика (кроме количества записанных элементов также дополнительно содержит минимальное и максимальное значения, сумма и среднее (для чисел); размер самой короткой строки и самой длинной (для строк)).
